## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Shashivardan-Jarpati/My-Professional-Portfolio.git
```

### Navigate to Project Folder

```bash
cd my-professional-portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
